# Future Ideas

## OpenClaw連携

- `AgentRunner` をOpenClaw CLIのラッパーにする
- OpenClawのtool call、diff、test resultをCompanionの会話ログへ流す
- タスク状態を `queued` / `running` / `blocked` / `done` で表示

## パーソナリティ

- `calm` / `senpai` / `buddy` などのプリセット
- ユーザーが苦手な声かけを無効化
- teaseは頻度上限を設け、集中を削らないようにする

## Focus Monitor

- 作業アプリの allowlist
- SNSや動画サイトへの切替をローカルルールで検知
- 25分集中、5分休憩の軽いポモドーロ
- VS Code、ブラウザ、ターミナルの文脈別スコア

## Memory

- 直近の完了タスク
- よく詰まるパターン
- ユーザーが好む声かけ

永続化する場合も、ローカル保存を基本にします。

## UI

- タスクキュー
- 現在のOpenClaw実行ステップ表示
- 集中タイムライン
- ステータスを1行ミニモードにする

## Safety

- 監視している項目を常に画面に表示
- 記録しないモードをデフォルトにする
- 叱責、羞恥、罪悪感を誘導する文言を禁止リスト化
